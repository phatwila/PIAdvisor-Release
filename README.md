# PIAdvisor – PixInsight Update Repository

Update feed for **PIAdvisor**, an AI-powered processing assistant for [PixInsight](https://pixinsight.com).

**Current release:** v1.0.3 · Windows x64 · PixInsight 1.8.9–1.9.x

## Install

1. Open PixInsight → **Resources → Updates → Manage Repositories → Add**
2. Paste the repository URL:

   ```
   https://phatwila.github.io/PIAdvisor-Release/
   ```

3. **Check for Updates** → Apply → Restart PixInsight

## Repository Contents

| Path | Purpose |
|:---|:---|
| `updates.xri` | Signed update manifest (read by PixInsight's updater) |
| `packages/` | Module ZIP archives |
| `docs/` | PIDoc documentation (also viewable inside PixInsight) |
